# Memory cards game

This is a second React project from The Odin Project curriculum. The goal of this project was to build a browser-based memory card game where the player has to find matching pairs.

The game fetches random dog images from an external API (thedogapi) and generates a shuffled board of cards. The player reveals two cards at a time and tries to match all pairs.

## 🚀 Features

- Dynamic card generation using an external API (useEffect hook)
- Fisher–Yates shuffle algorithm
- Memory-based gameplay
- Simple game logic with state management in React
- Custom UI with simple animations
- Click-lock system to prevent bugs
- Win detection with visual feedback

## 🛠️ Technologies

- React (with hooks)
- Vite
- CSS
- External API for images
